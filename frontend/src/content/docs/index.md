# Welcome to Autoform Documentation

## What is Autoform?

Autoform is a platform that simplifies deploying applications to AWS. Whether you're deploying personal projects or managing team infrastructure, Autoform handles the complexity of AWS for you.

### Key Features

- Creates and manages AWS infrastructure
- Builds and deploys from Git repositories
- Team collaboration features

## Prerequisites

Before you begin, you'll need:

- An Autoform account
- An AWS account with appropriate permissions
- A Git repository with your application code

## Quick Start

The most important step is setting up your AWS credentials. Once configured, you can deploy applications with just a Git URL.

[→ **AWS Setup Guide**](/docs/aws-setup) - Start here to configure your AWS credentials

Autoform creates AWS infrastructure for your deployments. Understanding the costs helps you plan your budget and optimize your applications.
